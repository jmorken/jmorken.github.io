Prime planetary gear1

ring gear teeth:
R = A002110(n) - A005867(n).

planet gear teeth:
P = A337022(n) = A002110(n)/2 - A005867(n).

sun gear teeth:
S = A005867(n).

For example for n=3:

R = A002110(3) - A005867(3)=30-8=22
P = A337022(3)=A002110(3)/2 - A005867(3)=7
S = A005867(3)=8

An example of this planetary gear for n=3 is on Thingiverse: https://www.thingiverse.com/thing:53451


      The planetary gear ratio is given by A038110/A060753 = (1/1, 1/2,
      1/3, 4/15, 8/35, 16/77 etc)

      

      Example for the gear ratio: S/(R + S) for S=8, R=22:
      8/(22+8)=8/30=4/15=A038110(4)/A060753(4).
